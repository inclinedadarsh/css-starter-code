about

This repository includes a CSS starter file and snippets for different Editors such as VS Code, Sublime Text and Atom. You can use these snippets to get a Head Start writing your CSS!
It includes some much-needed already-written code for margin, padding & box-sizing, font-family & font-size, color & background-color, and much more!
The Best Part about the code is it has Modern Variables!!! which keeps the code clean and make it easy for you to make the website responsive. Yes! It has media-queries too!

use

You can use the Snippets provided in VS Code, Sublime and Atom. To see a complete guide on usage, visit the GitHub Repository.
